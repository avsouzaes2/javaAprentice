## 🎯 Sobre o Projeto

Este repositório documenta minha jornada de aprendizado na linguagem Java. Aqui você encontrará:
- Exemplos de sintaxe básica e estrutura da linguagem;
- Exercícios práticos de lógica de programação;
- Conceitos de Orientação a Objetos (POO);
- Desafios e pequenas aplicações para fixação de conteúdo.

## 🚀 Como Executar os Projetos

Como o projeto segue a estrutura padrão Maven, você pode compilar e executar os exemplos de duas formas:

### Opção 1: Via Linha de Comando (Sem Maven instalado)
Navegue até a pasta da classe e use o `javac` para compilar e `java` para rodar:

```bash
# Navegar até a pasta da classe
cd src/main/java/com/avsouzaes/aprendendojava/Basico/

# Compilar o arquivo Java
javac Variables.java

# Executar a classe
java Variables
